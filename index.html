Find Out More

A scalable geotemporal visualization system you can install in your own cloud.

Geodashboard is a system for visualizing spatial and temporal data. It involves data being parsed from excel or csv files into organization of sensors, streams and datapoints. Datapoints are visualized as time series, depth graphs, stacked bar graphs or stacked line graphs depending on the nature of the data. The core software has been in development for the past five years with funding from a variety of sources. If you have any questions please email us or join us on HipChat.

Documentation

Built to be extensible

Visualizations for data can be easily customized. Data displayed on Geodashboard comes from different data sources, new parsers are written for each source using Pygeotemporal. Raw data obtained from the original sources is stored on Clowder.
